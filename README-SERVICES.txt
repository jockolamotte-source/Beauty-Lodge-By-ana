BEAUTY LODGE SERVICE MANAGER

Upload all files over the existing repository versions.

To change a service after upload:
1. Open the GitHub repository.
2. Click Actions.
3. Select Manage Website Services.
4. Click Run workflow.
5. Choose one service.
6. Choose Available, Temporarily unavailable, or Hidden.
7. Click Run workflow.

Available: the normal service card and Book button appear.
Temporarily unavailable: the card stays visible with an unavailable label and disabled button.
Hidden: the card disappears completely.

The action updates assets/services.json and commits the change automatically.
GitHub Pages normally refreshes shortly after the commit.
